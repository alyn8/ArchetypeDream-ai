# 🌙 Archetype AI: Jungian Dream Analyst

Bu proje, Carl Jung'un analitik psikoloji ekolünü modern yapay zeka teknolojileriyle birleştiren uçtan uca bir web uygulamasıdır. Kullanıcıların rüyalarını sembolik ve arketipler üzerinden analiz ederken, aynı zamanda bu rüyaları görselleştirir.

## 🚀 Özellikler

- **Derinlikli Rüya Analizi:** Groq API (LLama 3) kullanılarak, rüyalar Jungian arketipler ve sembolizm çerçevesinde analiz edilir.
- **Görselleştirme:** Pollinations AI entegrasyonu ile rüyanın atmosferine uygun yapay zeka görselleri üretilir.
- **Modern Arayüz:** Streamlit tabanlı, kullanıcı dostu ve estetik odaklı bir UI tasarımı.

## 🛠️ Teknolojiler

- **Dil:** Python 3.12
- **Framework:** [Streamlit](https://streamlit.io/)
- **LLM:** [Groq Cloud](https://console.groq.com/) (LangChain entegrasyonu ile)
- **Görüntü Üretimi:** [Pollinations.ai](https://pollinations.ai/)
- **Kütüphaneler:** `langchain-groq`, `Pillow`, `requests`
